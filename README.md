这个页面与其说是一个游戏，不如说是一个基于分形几何与群体智能算法的实时交互原型。它的核心机制和视觉风格都紧密围绕着“分形”这一概念展开。核心玩法：相位共振与生存。您需要操作一个由李萨如曲线动态生成的能量体，在三种颜色相位（蓝、红、黄）之间切换。当遇到庞大的Boids群体智能蜂群时，必须切换到与蜂群颜色相同的相位，才能“同化”它们并恢复生命值；如果相位错误发生碰撞，则会受到伤害。这本质上是参数匹配的博弈。交互与操作：游戏操作直观，但深度十足。使用 W/S 键控制实体在屏幕上下跃迁，躲避蜂群或主动冲入。按下 1、2、3 数字键切换相位，匹配蜂群颜色。界面左侧显示得分与生命值，右侧的相位指示灯会随切换动态变化。
